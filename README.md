# Android App Market: Analyzing Google Play Store Data

## Objective
The objective of this project is to analyze Google Play Store data to understand app market trends, category distribution, ratings behavior, pricing patterns, and user sentiment. The project aims to extract meaningful insights that help identify factors influencing app success in the Android ecosystem.

## Dataset
- apps.csv – Contains application details such as category, rating, installs, size, and price.
- user_reviews.csv – Contains user reviews and sentiment information.

## Steps Performed
- Loaded and explored Google Play Store datasets
- Cleaned data by correcting data types and handling missing values
- Analyzed category-wise app distribution
- Examined key metrics including ratings, installs, pricing, and size
- Performed sentiment analysis using user review data
- Created visualizations such as bar charts, histograms, scatter plots, and heatmaps to present insights

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Outcome / Results
- Identified dominant app categories contributing the highest number of applications
- Observed that the majority of apps are free with very few paid applications
- Analyzed relationships between installs, ratings, and pricing trends
- Found that positive sentiment dominates user reviews, indicating overall user satisfaction
